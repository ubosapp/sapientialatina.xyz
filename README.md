# Sapientia Latina

A simple web app for discovering and exploring famous Latin quotes.

## Deployment on Vercel

This project is configured for a fully automated deployment on Vercel. The `vercel.json` file contains all the necessary instructions to build the frontend application and deploy the serverless API function.

Simply connect your Git repository to Vercel, and it will be deployed automatically on every push. No manual configuration in the Vercel dashboard is required.
